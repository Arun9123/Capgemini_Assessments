# Capgemini Weekly Assessments

This repository contains various Java applications demonstrating different levels of software architecture.

---

## 1. Online Food Ordering System
**Location:** `/Weekly_Assessment_Feb28/Online Food Ordering System_OOPS`

A console-based application focused on **Object-Oriented Programming (OOP)** principles.

* **Key Features:**  Abstract classes and inheritance for Food Items.
    * Polymorphic list handling for Veg/Non-Veg orders.
    * Encapsulation of pricing & discount logic.
* **Visuals:** Output screenshots can be found in the same folder.

## 2. Employee Management System
**Location:** `/Weekly_Assessment_Feb28/Employee Management System_Hibernate`

A full-stack backend application demonstrating **Database Persistence** and **ORM (Object-Relational Mapping)**.

* **Technologies Used:** Java Core, Hibernate ORM, MySQL.
* **Key Features:**
    * Full CRUD (Create, Read, Update, Delete) operations for employees.
    * `hibernate.cfg.xml` configuration for automated table mapping.
    * Session management and Transaction handling.
* **Setup:**
    1. Ensure your SQL server is running.
    2. Update `hibernate.cfg.xml` with your local database credentials (username/password).
    3. Run `Launch.java` to initialize the SessionFactory.
